# 🤖 Nexugo — Python Discord Bot

**Nexugo** is a modular and customizable Discord bot built in Python using `discord.py`.  
It supports dynamic prefixes, embedded help commands, and an extensible cog‑based command system.

---

## 🚀 Features

- 📌 Dynamic prefixes per server stored in JSON  
- 🆘 Custom help command with clean embeds  
- 🧩 Modular design using cogs for scalable features  
- 🔐 Secure bot token handling via environment variables

---

## 🛠️ Tech Stack

- **Python**  
- **discord.py** (Discord bot API)  
- **JSON** for server config  

---

## 📁 Project Structure
```
nexugo/
├── cogs/             # Modular command extensions
├── main.py           # Entry point and bot setup
├── prefixes.json     # Stored prefixes per server
├── requirements.txt  # Python dependencies
└── runtime.txt
```
